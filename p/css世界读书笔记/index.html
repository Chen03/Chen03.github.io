Width & Height

  • width: auto 有四种表现：
    • fill-avaliable: block(<div> <p>)
    • fit-content: inline(<span> <a>)
    • min-content
    • max-content: CSS3新特性，不换行
  • height: auto 基本只有 fit-content 表现
  • 父元素为 inline，子元素为 block（互相依赖时）
